The overview page

Die Überblick Seite

Diese Seite soll einen groben Überblick über die aktuelle Lage bieten. Dazu werden folgende Ansichten angeboten:

This page gives you a rough overview about the current situation. The following views are offered:

  • Monatlicher Überschuss

    • Diese Ansicht zeigt die im aktuellen Monat vorhandenen Überschüsse (grün) bzw. Verluste (rot) an.

  • Kürzliche Ausgaben

    • Hier werden kürzlich getätigte Ausgaben aufgelistet.

  • Gesamtausgaben

    • Dieses Diagramm visualisiert alle bisher getätigten Ausgaben.

  • Kürzliche Einnahmen

    • Hier werden kürzlich generierte Einnahmen aufgelistet.

  • Gesamteinnahmen

    • Dieses Diagramm visualisiert alle bisher generierten Einnahmen.

  • Monthly surplus

    • This view displays the surplus (green) respectively loss (red) in the current month.

  • Recent spendings

    • This section lists all recent spendings.

  • All time spendings

    • This diagram visualizes the total amount of spendings done so far.

  • Recent earnings

    • This section lists all recent earnings.

  • All time earnings

    • This diagram visualizes the total amount of earnings generated so far.

You can download a monthly overview (for an arbitrary month) at the bottom of the balances page as a PDF-file.
